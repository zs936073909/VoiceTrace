## 简介

VoiceTrace 声迹是一款专为播音主持专业学生及语音训练需求者设计的桌面应用，集成了录音、智能分析、韵律评估、字级对齐、台风训练、文案写作、背诵训练等全流程训练工具。

## 核心功能

- 📝 **稿件管理** — 4 种分类（新闻播报/即兴评述/模拟主持/自定义），3 种语言，自定义语速标准
- 🎤 **专业录音** — WAV 16kHz 高质量录音，**空格键**一键标记卡顿，回放可快速跳转，支持删除录音
- 📊 **智能分析** — 自动计算语速、停顿、能量，判断是否符合行业标准；支持调用文本大模型生成个性化 AI 建议
- 🎬 **台风训练（v2.0 新增，v3.3 AI 增强）** — 基于 MediaPipe 的镜头感 + 肢体语言分析，支持站姿/坐姿双模式，6 维雷达图评分；新增 AI 实时点拨与整体总结，默认使用文本 LLM，可选多模态视觉分析
- 🧠 **背诵训练（v3.4 新增）** — 基于 FSRS-5 间隔重复算法 + 主动回忆 + 测试效应，支持演讲稿/主持词/应急话术三种场景，自动分段与关键词提取，三种训练模式（预览/提示/默写），智能复习调度与进度跟踪
- ✍️ **文案写作（v2.0 新增）** — 9 套专业模板 + AI 智能生成，兼容 OpenAI/DeepSeek/Moonshot/通义千问
- 🎧 **跟读模式（专业模式）** — 选示范录音 → 跟读 → AI 对比相似度，给出改进建议
- 🎵 **韵律分析（专业模式，v3.0 新增）** — 基于 Praat/Parselmouth 提取基频 F0、共振峰、强度、谐噪比 HNR、音调稳定性
- 🎯 **字级对齐（专业模式，v3.0 新增）** — 基于 faster-whisper 实现每个字/词的精确时间戳，标红漏读、慢读、快读
- 🤖 **实时 AI 陪练（专业模式，v3.2 新增）** — 边说边反馈，片段结束后 AI 教练给出点评
- 📈 **趋势追踪** — 多次录音的语速趋势图，可视化进步轨迹
- 📤 **数据导出（专业模式）** — 支持 CSV / JSON / PDF 三种格式

### v3.4 新增功能（背诵训练：基于 FSRS-5 间隔重复算法的科学记忆系统）

针对原产品中"助力背诵"功能弱化的问题，v3.4 引入完整的科学背诵训练系统，整合认知科学核心理论，覆盖演讲者与主持人两类核心场景。

#### 科学理论整合

- **艾宾浩斯遗忘曲线 + 间隔重复**：采用 FSRS-5（Free Spaced Repetition Scheduler）算法，基于记忆的可检索性（R）、稳定性（S）、难度（D）三变量模型动态调度复习间隔，比传统 SM-2 更精准
- **主动回忆 + 测试效应**：训练流程强制用户主动提取信息（默写模式），而非被动重读，研究证实此方式记忆巩固效率提升约 50%
- **工作记忆容量 + 组块化**：基于米勒 7±2 理论，自动将演讲稿按逻辑结构拆分为 60-90 秒的记忆组块
- **编码特异性原理**：支持模拟演讲环境（干扰背景音），在接近真实情境中编码以增强提取线索

#### 场景化功能

**演讲者场景**：
- 演讲稿分段记忆：按逻辑结构自动拆分（开场/主体/结尾）
- 关键词提示：可自定义提示密度（仅首字 / 关键词 / 首尾词）
- 节奏训练：结合内容熟悉度智能调整提示频率
- 模拟演讲环境：白噪声/咖啡馆/会场掌声等干扰背景音

**主持人场景**：
- 主持词快速记忆：优化短句与衔接词记忆，4 段一组
- 突发情况应对话术：建立应急内容库，2 段一组精熟训练
- 多角色对话模拟：支持分角色记忆

#### 三种训练模式

1. **预览模式**：完整展示段落 + 关键词高亮，适合首次学习
2. **提示模式**：仅显示关键词/首字/首尾词，引导主动回忆
3. **默写模式**：完全隐藏内容，用户口述后对照评分

#### FSRS-5 评分系统

每次复习后用户对记忆质量评分（1-4 分），算法据此调整下次复习间隔：

- **1 分（Again）**：完全忘记，重置稳定性，进入重学队列
- **2 分（Hard）**：勉强回忆，间隔缩短，难度增加
- **3 分（Good）**：正常回忆，按标准间隔延后
- **4 分（Easy）**：轻松回忆，间隔大幅延长，难度降低

#### 智能复习调度

- **到期提醒**：启动时自动检查今日待复习卡片数量
- **优先级队列**：到期卡片优先，新卡穿插，避免认知过载
- **长期跟踪**：记录每张卡片的复习历史、稳定性曲线、遗忘次数
- **统计仪表盘**：总卡片数、到期数、平均可检索性、按状态分布

#### 技术实现

- **调度引擎**：`core/memory_scheduler.py`，优先使用 `py-fsrs` 官方库，未安装时自动回退到内置 FSRS-5 简化实现
- **内容分析**：`core/content_analyzer.py`，基于规则 + NLP 的中英文分段、关键词提取、难度评估
- **数据持久化**：`memory_cards` 和 `review_logs` 两张表，支持完整 CRUD
- **可选依赖**：`jieba`（中文分词增强）、`py-fsrs`（官方调度器），未安装时自动降级

### v3.3 新增功能（AI 统一配置、台风训练增强与双模式界面）

- **文本模式（默认）**：只配置文本大模型，文案写作、语音分析建议、实时陪练点评、台风训练总结/实时点拨全部基于结构化数据调用文本 LLM。

- **AI 整体总结**：训练结束后基于完整数据生成整体评价 + 改进建议 + 针对性练习。
- **多模态画面分析**（仅多模态模式启用）：对训练画面进行视觉点评，指出体态、镜头感等具体问题。
- **模式状态提示**：界面上实时显示当前是「文本模式 / 多模态模式 / 本地规则模式」。

#### 语音分析 AI 建议

- 分析视图新增「获取 AI 建议」按钮，基于语速、卡顿、韵律、字级对齐等结构化数据调用文本 LLM 生成个性化训练反馈。

## 快速开始

### 方式一：一键启动（推荐普通用户）

1. 前往 [Releases](../../releases) 下载最新的 `VoiceTrace-v3.4-portable.zip`
2. 解压到任意目录
3. 双击 `VoiceTrace.exe` 即可运行，无需安装 Python

### 方式二：从源码运行（推荐开发者）

```bash
# 1. 克隆仓库
git clone https://github.com/zs936073909/VoiceTrace.git
cd VoiceTrace

# 2. 安装依赖
pip install -r requirements.txt

# 3. 运行
python main.py
```

**系统要求**：Python 3.10+，Windows 10/11 64 位

### 从源码打包成可执行文件

```bash
# 1. 安装打包依赖（requirements.txt 已包含 pyinstaller）
pip install -r requirements.txt

# 2. 执行打包脚本
python build_windows.py

# 3. 输出位置
# dist/VoiceTrace/VoiceTrace.exe
```

打包完成后，`dist/VoiceTrace` 就是便携版文件夹，可直接压缩分发。

## v3.0 新增功能详解

### 韵律分析

基于 Praat 声学分析引擎，对录音进行专业级韵律评估：

- **基频 F0 曲线**：反映音高变化，可用于诊断语调平淡或过度起伏
- **共振峰 F1/F2/F3**：反映元音音色和口腔共鸣位置
- **强度曲线**：反映声音强弱变化，评估重音和情感表达
- **谐噪比 HNR**：衡量声音清晰度，数值越高越"干净"
- **声调稳定性/声调得分**：针对中文普通话的声调质量评估

### v2.0 功能回顾

#### 台风训练

通过摄像头实时分析你的**镜头感**和**肢体语言**：

| 维度 | 说明 |
|---|---|
| 眼神交流 | 虹膜追踪，判断是否直视镜头 |
| 表情管理 | 微笑度 + 紧张度识别 |
| 头部姿态 | yaw/pitch/roll 三维估计 |
| 站姿/坐姿 | 站姿检测倾斜，坐姿额外检测探颈 |
| 手势 | 手腕运动幅度分析 |
| 稳定性 | 身体晃动幅度 |

支持**站姿**和**坐姿**两种模式。

#### 文案写作

- **模板生成**：9 套专业模板（新闻播报、即兴评述、模拟主持、演讲），离线可用
- **AI 智能生成**：支持任何兼容 OpenAI 格式的 API（DeepSeek、Moonshot、通义千问等）
- **保存为稿件**：一键导入稿件管理，方便后续录音练习
